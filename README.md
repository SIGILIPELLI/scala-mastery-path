# Scala Mastery Path

A free, structured, module-wise Scala training program — entry level to master level, with runnable code in every module and a hands-on project at the end of each level.

**Live site:** https://sigilipelli.github.io/scala-mastery-path/

## Contents

- **Level 1 · Entry** — language foundations
- **Level 2 · Intermediate** — idiomatic, functional Scala
- **Level 3 · Advanced** — concurrency, APIs, databases
- **Level 4 · Master** — production systems & architecture

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
